## TROUBLESHOOTING

### Page shows 404
```
→ Wait 2-3 minutes (GitHub takes time)
→ Clear browser cache (Ctrl+Shift+Del)
→ Check GitHub Pages is enabled in Settings
```

### Page loads but looks broken
```
→ Try incognito/private window
→ Clear cache and reload
→ Check browser console (F12)
→ Verify index.html was uploaded correctly
```

### Can't find GitHub Pages settings
```
→ Make sure repo is PUBLIC
→ Go to Settings (not repo main page)
→ Look in left sidebar for "Pages"
→ Must be repo owner
```

### Analysis not working
```
→ Test locally in index.html first
→ Check browser console for errors
→ Try sample data (all fields optional)
→ Reload page completely
```
