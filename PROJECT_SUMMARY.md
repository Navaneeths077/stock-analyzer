## 💡 How It Works

### 1. User Inputs Metrics
User fills in company financial data (all fields optional)

### 2. System Calculates Scores
```
Valuation Score = P/E + P/B + Dividend + EPS
Profitability Score = ROE + ROCE + Margins
Growth Score = Revenue Growth + CAGR + Profit Growth
Health Score = D/E + Current Ratio + Reserves + Ownership
```

### 3. Overall Rating
```
Overall = (Valuation + Profitability + Growth + Health) ÷ 4
```

### 4. Generate Recommendation
```
75+ = STRONG BUY (Green)
60-74 = BUY (Blue)
45-59 = HOLD (Yellow)
30-44 = WEAK SELL (Orange)
0-29 = SELL (Red)
```

### 5. Display Results
- Overall score with visual
- Category breakdown
- 8 key metrics with ratings
- Strengths & concerns list
